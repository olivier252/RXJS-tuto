L'operator map applique une fonction sur chaque élémentde l'observable. Comprend 2 pramètres :

  • project : la fonction à appliquer sur chaque valeur émise par l'observable. Le paramètre index "i" est le nombre pour la énième émission qui est survenue après la souscription.
  • this.Arg : optionnel (undefined par défaut) définit "this dans project fonction
